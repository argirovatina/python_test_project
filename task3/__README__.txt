Задача №3.

Дан файл с исходным кодом, реализующим упрощенную модель логики магазина,
который позволяет игрокам покупать танки и орудия к ним (файл TankShop.py).

Необходимо:
1. Разобраться в логике работы кода.
2. Составить тест кейсы для проверки методов.
3. Покрыть тестами 2 соответствующих метода магазина.
4. Менять исходники файла нельзя.
5. Получить по итогу: 
    * Файл, который можно положить около файла TankShop,
      запустить с помощью python и получить результаты проверки.
    * Файл с описанием найденных ошибок и тесткейсов.
